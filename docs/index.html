DOWNSTAIRS

downstairs is a roguelike video game created by Daniel Nimmervoll and Lukas Singer. Our goal is to write, play, maintain and publish a classic roguelike.

You can Download an executable for your favourite operating system from here.

Source code and build instructions can be found here.

If you found a bug or have ideas for improovements feel free to open an Issue on github or send an email to .

If you really like downstairs and want to support us you can donate to our PayPal account or to one of the tools or websites we use.
